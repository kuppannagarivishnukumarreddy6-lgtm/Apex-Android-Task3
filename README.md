# Student Registration App with API Integration

## 📌 Project Overview

This Android application was developed as part of the ApexPlanet Android Development Internship - Task 3 (Backend Integration and API Development).

The app allows users to enter student details, navigate between screens, and interact with REST APIs using Retrofit.

## ✨ Features

* Student Registration Form
* Name, Email, and Phone Number Input
* Navigation Between Screens
* REST API Integration using Retrofit
* GET Request (Fetch User Data)
* POST Request (Create User)
* PUT Request (Update User)
* DELETE Request (Delete User)
* Error Handling and Response Handling
* Real Device Testing

## 🛠️ Technologies Used

* Kotlin
* Android Studio
* Jetpack Compose
* Retrofit
* JSONPlaceholder API
* Git & GitHub

## 📡 API Endpoints Used

* GET /users/1
* POST /users
* PUT /users/{id}
* DELETE /users/{id}

Base URL:
https://jsonplaceholder.typicode.com/
